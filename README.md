# QL-Assembler-Disassember
Contains the assembler and disassembler listings from the book Quick QL Machine Language by Alan Giles, 1984 Melbourne House.
These files are scanned and ORCed by AI. 

## Status
- Uploaded in raw state
- Working through the code to remove any OCR errors and try to correct whitespace/indentation (not critical for Superbasic)
